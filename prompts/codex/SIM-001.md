本项目研究对象是：在强非线性动力学和扰动不确定条件下，多模态水下机器人的学习型 MPC / RL-MPC 控制方法。

## 要求

1. 实现类 `PlanarUnderwaterVehicle3DOF`。
2. 状态向量：

```text
[x, y, psi, u, v, r]
其中：
x, y：惯性系位置
psi：偏航角
u, v：体坐标系纵向和横向速度
r：偏航角速度
```

3. 控制向量：

```text
[tau_u, tau_r]
其中：
tau_u：纵向推力
tau_r：偏航力矩
```

4. 模型参数需支持配置：

- 类质量参数
- 偏航惯量参数
- 线性阻尼
- 二次阻尼
- 可选扰动项

5. 至少实现以下接口：

```python
def dynamics(self, state, control, disturbance=None) -> np.ndarray:
    ...


def step(self, state, control, dt, disturbance=None) -> np.ndarray:
    ...
```

6. 仅使用 NumPy，不要引入重依赖。
7. 代码保持清晰、简单，便于后续 MPC / RL 实验复用。

## 测试要求

请编写测试，至少验证以下内容：

- `dynamics` 返回形状 `(6,)`
