# Task SIM-002: Implement PID Baseline Controller

You are Codex, a local coding agent working in this Git repository.

## Goal

Implement a simple PID baseline controller for early simulation comparisons.

## Research Context

This project needs a lightweight baseline controller before extending to learning-based MPC and RL-MPC variants.

## Files

- Create or modify: `src/controllers/pid.py`
- Create or modify: `tests/test_pid.py`

## Requirements

1. Implement a class `PIDController`.
2. Support proportional, integral, and derivative gains.
3. Support simple position-error-to-control output.
4. Support output saturation with configurable lower and upper bounds.
5. Expose a clear update interface for repeated simulation calls.
6. Keep the implementation dependency-light and readable.

## Tests

Create tests that verify:

- controller output is deterministic for fixed inputs;
- zero error produces zero output when integral state is zero;
- saturation limits are enforced;
- integral term accumulates across repeated calls;
- derivative action responds to error change.
